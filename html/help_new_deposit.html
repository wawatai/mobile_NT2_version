存款

  • 如何存款到TB588帳戶?

    登入會員帳號後，點選畫中間的【存款】圖示，即可進入存款頁面，依據不同存款方式進行存款。

    提供何種存款方式

    1. 銀行卡轉帳

    2. 超商代碼繳費

    3. ATM匯款

    4. USDT

  • 銀行卡轉帳-步驟教學

    1. 至存款畫面選擇【銀行卡轉帳】

    2. 選擇您將進行轉帳的銀行帳戶，並填寫預計存款金額

    3. 點選立即存款

    4. 成功繳費後，即可至【我的錢包】確認入帳金額

  • 超商代碼繳費-步驟教學

    1. 至存款畫面選擇【超商代碼】

    2. 選擇通道，並填寫預計存款金額

    3. 點選立即存款，紀錄超商代碼至超商繳費。或點選畫面中央【手機點此進入快速繳費介面】，並選擇預繳費超商圖示，即可以顯示的條碼於超商進行繳費。

    注意:條碼產生後，有效期限為10分鐘
    請到達超商門市時，再點選下方按鈕

    4. 成功繳費後，請妥善保存小白單至少一個月。

  • ATM匯款-步驟教學

    1. 至存款畫面選擇【ATM】

    2. 選擇通道可任意選擇，並填寫預計存款金額

    3. 點選立即存款

    4. 依據畫面提供的虛擬帳戶進行轉帳

    5. 至【我的錢包】確認入帳金額

  • USDT-步驟教學

    1. 至存款畫面選擇【USDT】

    2. 選擇通道可任意選擇，並填寫預計存款金額。存款金額為USDT幣值，同步顯示可入金金額。

    3. 點選立即存款

    4. 依據畫面提供充幣地址進行轉帳

    5. 至【我的錢包】確認入帳金額
