  • 如何註冊帳號

    1. 至官網首頁，點選【登入/註冊】

    2. 點選【註冊新會員】

    3. 輸入【帳號、密碼、手機】，並使用手機收取簡訊驗證碼驗證，邀請碼欄位若無則無需填寫，勾選同意條款，最後點選確認

  • 在網站註冊之後，我的個人資料信息安全有保障嗎？

    關於客戶帳戶資料安全，以及客戶隱私權，請參考隱私政策。

  • 忘記已註冊的用戶名怎麽辦？

    若忘記帳號，您可以聯繫我們在線客服團隊，為您提供必要協助。

  • 登錄帳戶提示被鎖是什麼原因？

    若會員輸入錯誤密碼5次以上，為了保障會員帳號的安全，系統自動鎖定會員帳號，出現此類問題，可以聯繫在線客服團隊進行溝通解除鎖定。

  • 我可以申請註銷賬戶嗎？

    暫時不接受任何理由註銷會員帳戶，您可以與在線客服聯繫申請凍結/停用您的會員帳戶。

  • 忘記密碼怎麼辦？

    1.至官網首頁，點選【登入/註冊】

    2. 點選【忘記密碼】

    3. 輸入註冊手機號碼，點選重置密碼

    4. 輸入簡訊驗證碼、新密碼，點選重置密碼

    5. 顯示更改完成，點選確認後，即可使用新密碼登入

  • 如何修改帳戶密碼？

    1. 至官網首頁，登入帳號後點選【會員中心】

    2. 點選帳號欄位旁的【個人資料】

    3. 進入【個人資料】往下滑動至【修改密碼】點選【修改】

    4. 輸入舊密碼及新密碼後點選【確認】

    5. 顯示更改完成，點選確認後，即可使用新密碼登入

  • 存款教學-超商繳費

    1. 至官網首頁，登入帳號後點選【存/轉/提】

    2. 點選【存款】

    3. 支付方式選擇【超商代碼】

    4. 選擇通道欄位點選【超商支付】，選擇超商欄位選擇【欲前往繳費的超商】，最後輸入【存款金額】，點選【立即存款】

    5. 顯示更改完成，點選確認後，即可使用新密碼登入

  • 存款教學-虛擬帳戶

    1. 至官網首頁，點選【存/轉/提】

    2. 點選【存款】

    3. 支付方式選擇【ATM】

    4. 選擇通道欄位點選【虛擬帳號】，最後輸入【存款金額】，點選【立即存款】

  • 存款教學-銀行ATM

  • 存款教學-銀行卡轉帳

    1. 至官網首頁，點選【存/轉/提】

    2. 點選【存款】

    3. 支付方式選擇【銀行卡轉帳】，選擇要使用的【轉出銀行】、【存款金額】，點選【立即存款】

  • 如何提款帳戶內資金

    1. 至官網首頁，點選【存/轉/提】

    2. 點選【提款】

    3. 選擇【提領銀行】以及【提領金額】後點選【確認送出】，審核完成並出款後將會發送站內信通知

    4. 查詢提款審核狀態，可至【會員中心】點選【交易紀錄】

    5. 類型選擇【提款】點選查詢時間後，點擊【查詢】

    6. 狀態欄位顯示【審核中】代表該體提款正在僅行審核，顯示【通過】則表示出款完成

  • 提款常見問題

    Q:可以使用別人的銀行卡進行提款嗎？

    A:爲了確保客戶的賬戶資金安全,提款需要使用遊戲,且提款銀行卡姓名必須與註冊姓名一致。

  • 申請提款注意事項

    提款綁定的銀行卡姓名需與遊戲帳戶的註冊姓名一致

    提款綁定的銀行卡信息需要正確。

    若您未申請任何紅利優惠,投注滿一倍流水即可申請提款。

    若申請首存紅利,則需要滿足優惠活動寫明的有效投注額要求。

    全天24小時都可進行提款申請。

    單一綁定帳戶，最高申請提款額度為49萬。

    提款50萬(含)以上，請綁定兩個銀行帳戶。提款100萬(含)以上，請綁定三個銀行帳戶。依此類推，提款金額將分配至各別帳戶中。

  • 遊戲帳戶裡有錢為什麼無法提款？

    您需要先將資金從遊戲平臺轉至中心賬戶後才能進行提款操作。

  • 可以綁定多張銀行卡嗎？

    會員帳戶可以綁定多張銀行卡進行提款操作,在帳戶中心選擇 【 銀行卡管理 】,進行新增添加即可。

  • 提款爲什麽還需要審核？

    提款審核是相關部門在給您辦理出款之前的一個簡單的核實步驟,是為了確保客戶資金安全,所以需要核實相關信息。

  • 如何轉帳?

    1. 至官網首頁，點選【存/轉/提】>【轉帳】

    2. 選擇要轉移的館別，輸入轉點金額後點選【確認轉帳】即可
    ※點選【一鍵回收】，可以將所有館別的點數，全數轉回至中心錢包

  • 交易紀錄查詢

    1. 至官網首頁點選【會員中心】>【交易紀錄】

    2. 點選欲查詢類型

    3. 設定查詢時間，點選【查詢】

    4. 向下滑動擊可查閱紀錄

  • 投注紀錄查詢

    1. 至官網首頁點選【會員中心】>【投注紀錄】

    2. 設定查詢的【平台】、【時間】後，點選【確認送出】

    3. 顯示投注紀錄後點選【展開】

    4. 向下滑動擊可查閱【投注明細】

  • 站內信教學

    1. 至官網首頁，點選【會員中心】>【站內信】
    ※若有顯示信件數量，代表有未讀取信件

    2. 點選信件後，可以查閱完整內容，右上角的鉛筆圖案，可以將信件設為已讀或是刪除

    3. 打開信件後，閱讀完成，如無須留存，點選右上方【刪除】即可

  • 如何綁定銀行卡

    1. 至官網首頁，點選【會員中心】>【綁定銀行卡】

    2. 系統會自動跳出填寫個人資料的提示，點擊【同意】即可跳轉填寫畫面

    3. 填寫【真實姓名】【生日】，【LINE】或【微信】擇一填寫即可，填寫完成後，點選保存資料

    4. 回到【綁定銀行卡】，銀行卡類別選擇【銀行卡】，輸入【銀行名稱】、【分行名稱】

    5. 輸入【銀行帳號】，並於【確認帳號】再次輸入銀行帳號，填寫完成後，點擊【確認】

    6. 顯示新增成功就完成囉
